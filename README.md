## 关于vha-appDemo

vha-components + vha-native 的App演示程序。[apk下载](https://github.com/neoStudioGroup/vha-appDemo/releases)

## 如何使用

运行：

```shell
npm install
npm run serve
```

编译生成：

```shell
cordova platform add android
cordova build android
```
